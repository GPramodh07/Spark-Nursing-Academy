# Spark Nurses Academy (Prototype)

This is a prototype project for a nursing exam preparation platform.
It is an early version and will be developed further.

## What is available now

- Home page UI
- Registration and login pages
- Basic auth API (register/login)
- MySQL connection

## Tech used

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MySQL

## Run locally

1. Install packages

```bash
npm install
```

2. Create MySQL database

```sql
CREATE DATABASE sparks;
```

3. Update DB details in [config/db.js](config/db.js)

4. Start server

```bash
npm run test
```

5. Open in browser

- http://localhost:5000/pages/home/home.html

## Notes

- This is not production-ready yet.
- Admin and user dashboards are still pending.
- OTP backend verification is not complete yet.
